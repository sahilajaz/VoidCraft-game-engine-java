🌌 VoidCraft

VoidCraft is a custom-built game engine written in Java, powered by OpenGL through LWJGL.

It is designed to explore and implement modern real-time rendering techniques while maintaining full control over the graphics pipeline and engine architecture.

VoidCraft focuses on understanding how a game engine works internally — from managing GPU buffers and shaders to handling transformations, camera systems, and rendering workflows. Instead of relying on high-level frameworks, the engine interacts directly with OpenGL, giving fine-grained control over performance and rendering behavior.

The engine is structured to be modular and scalable, making it suitable for experimenting with rendering systems, scene management, and core engine components. Every system is built with clarity and learning in mind, emphasizing clean design and efficient communication between CPU and GPU.

VoidCraft represents building something powerful from nothing — crafting entire virtual worlds starting from raw vertices and shaders.
